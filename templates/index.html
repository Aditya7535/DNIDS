System Architecture

Distributed topology powered by Apache Storm

KDD Spout

Reads the KDD Cup 99 dataset line-by-line. Each record contains 41 network traffic features plus a label for classification.

Python • CSV Reader • Data Source

Preprocess Bolt

Encodes categorical features using LabelEncoder and scales all 41 numerical features using StandardScaler for ML readiness.

scikit-learn • StandardScaler • LabelEncoder

ML Bolt

Pre-trained DecisionTreeClassifier predicts each connection as Normal (0) or Attack (1) with confidence scores.

Decision Tree • Binary Classification • joblib

Alert Bolt

Generates real-time intrusion alerts with attack type identification, tracks accuracy metrics, and logs all detections.

Alerts • Statistics • Logging
